Flow

接入步骤

  1. 先在问答管理的“API 密钥”标签页里为当前场景创建一个密钥。
  2. 调用场景信息接口，拿到 name、description 和 ui_config 初始化页面。
  3. 再向流式问答接口发送 messages 数组；服务端会基于当前场景绑定的知识片段做检索增强。
  4. 流中会依次返回 token 和 citations，最后用 [DONE] 结束整个响应。
