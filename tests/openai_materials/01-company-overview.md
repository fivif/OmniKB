# OpenAI 公司概述

## 公司信息
- **公司名称**: OpenAI
- **总部**: 美国加利福尼亚州旧金山
- **CEO**: Sam Altman
- **成立时间**: 2015年（非营利）, 2019年（转型 capped-profit）
- **主要融资**: Microsoft 数十亿美元投资
- **估值**: ~$3000亿（2025-2026融资轮）
- **关键里程碑**:
  - 2015: 成立
  - 2020: GPT-3 发布
  - 2022.11: ChatGPT 发布
  - 2023.03: GPT-4 发布
  - 2023.11: GPTs, GPT-4 Turbo, Assistants API
  - 2024.02: Sora 发布
  - 2024.05: GPT-4o 发布
  - 2024.07: GPT-4o-mini
  - 2024.09: o1 推理模型
  - 2025.01: o3-mini
  - 2025.04: GPT-4.1 系列
  - 2025-2026: GPT-5 系列, o4-mini, Sora 2

## ChatGPT 消费者产品
- **Plus**: ~$20/月, 包含 GPT-4o, DALL-E 3, 数据分析, 文件上传, GPTs, Canvas
- **Pro**: ~$200/月, 更高速度, 优先访问新模型
- **Team**: ~$25/人/月, 团队管理
- **Enterprise**: 自定义定价

## 功能要点
- GPTs: 自定义 ChatGPT（2023年11月推出）
- Canvas: 写作/代码协作编辑
- Web Browsing: 搜索功能
- Data Analysis: 代码解释器
- DALL-E 集成: 图像生成
