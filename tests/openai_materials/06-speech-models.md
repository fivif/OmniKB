# OpenAI 语音模型

## Whisper (语音转文字)
- 模型: whisper-1
- 定价: $0.006/分钟（约$0.006/1M tokens）
- 支持格式: flac, mp3, mp4, mpeg, mpga, m4a, ogg, wav, webm
- 功能: 多语言识别、翻译、语言识别、时间戳

## GPT-4o Transcribe (增强型语音转文字)
- 输入音频 $2.50/1M tokens
- 输出文本 $10.00/1M tokens
- 比 Whisper 更高的准确性

## GPT-4o-mini Transcribe
- 输入 $1.25/1M tokens
- 输出 $5.00/1M tokens

## TTS / TTS-HD (文字转语音)
- tts-1: 低延迟
- tts-1-hd: 更高音质
- gpt-4o-mini-tts: 最新, 支持指令控制语气和情绪
- 定价: $15.00/1M 字符
- 声音选项: alloy, echo, fable, onyx, nova, shimmer, coral, marin, cedar, ash, ballad, sage, verse（共13种）
- 推荐最佳音质: marin, cedar
- 输出格式: mp3(默认), opus, aac, flac, wav, pcm
- 速度范围: 0.25 ~ 4.0
- gpt-4o-mini-tts 最大输入: 2000 tokens
